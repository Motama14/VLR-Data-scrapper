# VLR Data Scrapper

### What can you pull from vlr using the scripts inside this repo?

---
##### With v0.1:
scrapcomps.py -> Scraps all the recent comps that a team has been using on all maps

scrapcompsv2.py -> Is able to put all the comps in a csv file along with the map

resultscrapper.py -> Gets all the results of different matchs inside the same event

scrapteams.py -> Gets all the teams that participates in an event
